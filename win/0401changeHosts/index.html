加速github访问

  1. 打开Powershell,切换至C:\windows\system32\drivers\etc

    Windows PowerShell

PS C:\WINDOWS\system32> cd C:\Windows\System32\drivers\etc

  2. 执行以下命令，即可以管理员权限用记事本打开hosts文件：

    PS C:\Windows\System32\drivers\etc> notepad hosts

C:\Windows\System32\drivers\etc> cmd

  3. 添加ip地址：

    13.250.177.223 github.com
192.30.252.131 github.com
192.30.253.113 github.com

74.125.237.1    dl-ssl.google.com
74.125.128.95   ajax.googleapis.com
151.101.88.249  github.global.ssl.fastly.net
173.194.127.200 groups.google.com
185.31.16.185   github.global.ssl.fastly.net

  1. 保存后，刷新：

    C:\Windows\System32\drivers\etc> ipconfig /flushdns

Windows IP 配置
已成功刷新 DNS 解析缓存。

重新打开github，访问速度明显提升！
