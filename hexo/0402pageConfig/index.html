首页文章预览显示

发布文章后，网站首页默认全文显示。但文章太长，会占用过多布局。这时就需要将文章设为预览样式。

NexT主题下配置文件 _config.yml 按如下配置即可生效：

# Automatically Excerpt. Not recommend.
# Please use <!-- more --> in the post to control excerpt accurately.
auto_excerpt:
  enable: false
  length: 150

将enable: true，即可是预览生效；而 length 表示预览显示文字长度。

除此之外，还有另外一种配置方法。在要预览显示的文章内增加一句：<!--more--> 即可使这一句以上的文字成为预览项。

代码部分主题样式

  1. 站点配置文件：_config.yml

    highlight:
  auto_detect: false

    需配置自动检测 auto_detect 为 true。

  2. 主题配置文件：_config.yml

    # Code Highlight theme
# Available value:
#    normal | night | night eighties | night blue | night bright
# https://github.com/chriskempson/tomorrow-theme
highlight_theme: night eighties
