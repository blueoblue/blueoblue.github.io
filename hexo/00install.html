hexo配置

hexo博客系统搭建

需安装node.js、git，并添加环境变量。若git官网下载过慢或打不开可前往淘宝镜像源：https://npm.taobao.org/mirrors/git-for-windows/ 来下载。

关于环境变量管理方式见文末。

一、安装

  1. 新建一个空目录：blog，并切换至该目录下。

  1. 首先安装hexo模块

    其中npm安装模式分为：

    全局安装：npm install **-g** module

    即添加 -g 选项。

    局部安装：npm install module

    即不添加 -g 选项。

  2. 新建网站

    需将hexo-cli模块的可执行文件加入环境变量中：C:\Users\pc-username\AppData\Roaming\npm\node_modules\hexo-cli\bin

    若未指定 目录folder ，则默认在当卡目录下初始化网站。

  3. 安装依赖、文件部署

  4. 启动服务

    浏览器中键入：”localhost:4000”。

二、github仓库设置

创建新仓库：github-username.github.io

点击该仓库最右侧：setting，找到 GitHub Pages 选项>

选择网页主题样式。

完成后，搭建完成。

三、环境变量管理器

配置开发环境时，经常需要添加相应环境变量。但设置比较复杂，所以推荐一款管理器，来轻松管理环境变量。

Rapid Environment Editor ,软件支持中文，并且免费。无广告。

分为左侧系统变量，右侧用户变量。修改变量是，需点击变量左侧小“+”展开后，在修改。而修改系统变量时，需点击中间分割线上方的“以管理员身份启动”。修改完后，点击保存。

若要使环境变量生效，不必重启系统。只需在“任务管理器”中 重启 “Windows资源管理器” （即文件管理器），新设环境变量便能生效。
